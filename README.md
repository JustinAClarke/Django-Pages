## Django Pages

### Problem

Required a bare basic cms, for use with some very very basic pages,
Raw html codeing, with a possible WYSIWYG editor.

### Solution

Build a django app that has a page module which contains;
 * A Title
 * A Url (slug)
 * Some HTML Content
